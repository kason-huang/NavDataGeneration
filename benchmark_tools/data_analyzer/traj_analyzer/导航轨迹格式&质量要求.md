# 轨迹问题汇总
备注（√表示是问题，×表示不是问题，？表示不确定，高中低表示优先级）

* [√高]起始action不是STOP
* [√高]终点action不是STOP
* [?]轨迹长度超出500

* [√高]探索路径复杂
* [√]同一轨迹下存在连续低头的视角
* [√]视野内存在目标类别，但未走过去
* [√]经过viewpoint，但是没有成功找到
* [×]整段轨迹存在上下楼
    任务不涉及爬楼
* [×]轨迹中存在LOOK_UP，LOOK_DOWN
    动作空间没有LOOK_UP, LOOK_DOWN
* [?]停位时视野内无目标 or 占比较低
* [?]频繁LOOK_UP, LOOK_DOWN, TURN_LEFT, TURN_RIGHT
* [?]在中途停止不动大概3秒，过好一会再动起来
* [?]起点附近有目标类别，但是没有360环视，导致错过
    * 探索策略的选择相关
* [?]同楼层存在目标类别，但是找到了其他楼层的
    * 探索策略的选择相关
* [?]路过门口没有进行探索
    * 探索策略的选择相关
